## walker

### 项目背景

##### nodejs实现一个websocket server。

### 技术方案

* 规范：
    * ES6
    * CommonJS
* 服务端技术结构
    * koa
    * koa.io
### 测试demo
```
npm install -l
node -harmony server
```

### 关于项目规范

* 版本发布
    * git版本为dev/0.0.1 dev-develop版 0.0.1 第一个数字是大版本，第二个数字是正式迭代版本，第三个数字是每次的小版本。
    * 版本原则，大版本只有到项目改版使用 正式版本只要有发布都是正式版本 小版本为开发版本多人协作等等。
    * 一旦发布正式版，需要git tag pub/0.0.1

* 测试
    * npm install -l
    * node -harmony server
